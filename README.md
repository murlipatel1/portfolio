# Murli Patel — Portfolio

Recruiter-first personal site for **Murli Patel**, Full Stack Developer at Kaushalam Digital.

Built with **Vite 8**, **React 19**, **TypeScript**, **Tailwind CSS v4**, and **Motion**.

## Requirements

- **Node.js 24.x** (LTS)
- npm 10+

```bash
# with nvm
nvm install 24
nvm use 24
```

## Develop

```bash
npm install
npm run dev
```

## Build & deploy (GitHub Pages)

```bash
npm run build
npm run deploy
```

Optional EmailJS (otherwise the contact form opens mailto):

```env
VITE_EMAILJS_SERVICE_ID=
VITE_EMAILJS_TEMPLATE_ID=
VITE_EMAILJS_PUBLIC_KEY=
```
